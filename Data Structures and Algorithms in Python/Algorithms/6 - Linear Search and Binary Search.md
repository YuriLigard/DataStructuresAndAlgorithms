# Linear Search 
- Looping through each element 
- Element found 
	- algorithm stops 
	- returns the result
- Element not found
	- algorithm continues


![Screenshot from 2024-02-08 13-48-32.png](../../../_resources/Screenshot%20from%202024-02-08%2013-48-32.png)
- **Complexity:** O(n)
